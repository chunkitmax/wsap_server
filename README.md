# WSAP_Server

## Setup
```bash
# make sure this file has execute permission
$ chmod +x start_docker.sh
# install all dependencies
$ npm install
```

## Run
```bash
# debug (w/ VNC support port 5900)
$ ./start_docker.sh debug
# or
$ ./start_docker.sh

$ npm start
```
